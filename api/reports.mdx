## Dashboard Summary

<CodeGroup>
```bash GET /reports/dashboard-summary
curl -X GET "http://localhost:8080/api/v1/reports/dashboard-summary" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</CodeGroup>

Get high-level summary metrics for the dashboard.

### Response

<ResponseField name="data" type="object">
  <Expandable title="DashboardSummaryResponse">
    <ResponseField name="total_sales" type="number" required>
      Total sales amount
    </ResponseField>
    
    <ResponseField name="total_orders" type="integer" required>
      Total number of orders
    </ResponseField>
    
    <ResponseField name="unique_cashier" type="integer" required>
      Number of unique cashiers
    </ResponseField>
    
    <ResponseField name="total_products" type="integer" required>
      Total number of products
    </ResponseField>
  </Expandable>
</ResponseField>

### Roles

Accessible by: `admin`, `manager`, `cashier`

## Sales Reports

<CodeGroup>
```bash GET /reports/sales
curl -X GET "http://localhost:8080/api/v1/reports/sales?start_date=2024-01-01&end_date=2024-01-31" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</CodeGroup>

Get aggregated sales data grouped by date within a specified range.

### Query Parameters

<ParamField query="start_date" type="string" required>
  Start date in YYYY-MM-DD format
</ParamField>

<ParamField query="end_date" type="string" required>
  End date in YYYY-MM-DD format
</ParamField>

### Response

<ResponseField name="data" type="array">
  Array of daily sales summaries
  
  <Expandable title="SalesReport">
    <ResponseField name="date" type="string" required>
      Date of the sales data
    </ResponseField>
    
    <ResponseField name="order_count" type="integer" required>
      Number of orders on this date
    </ResponseField>
    
    <ResponseField name="total_sales" type="number" required>
      Total sales amount for this date
    </ResponseField>
  </Expandable>
</ResponseField>

### Roles

Accessible by: `admin`, `manager`, `cashier`
