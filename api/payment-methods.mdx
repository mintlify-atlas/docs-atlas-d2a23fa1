## List Payment Methods

<CodeGroup>
```bash GET /payment-methods
curl -X GET "http://localhost:8080/api/v1/payment-methods" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</CodeGroup>

Get a list of all active payment methods (e.g., Cash, QRIS, Gopay).

### Response

<ResponseField name="data" type="array">
  Array of payment method objects
  
  <Expandable title="PaymentMethodResponse">
    <ResponseField name="id" type="integer" required>
      Payment method ID
    </ResponseField>
    
    <ResponseField name="name" type="string" required>
      Payment method name (e.g., "Cash", "QRIS")
    </ResponseField>
    
    <ResponseField name="is_active" type="boolean" required>
      Whether the payment method is currently active
    </ResponseField>
    
    <ResponseField name="created_at" type="string" required>
      Timestamp when payment method was created
    </ResponseField>
  </Expandable>
</ResponseField>
