## List Products

<CodeBlock title="GET /api/v1/products">
Get a list of products with filtering by category and search term.

**Authentication Required:** Yes

**Roles:** `admin`, `manager`, `cashier`
</CodeBlock>

### Query Parameters

<ParamField query="page" type="integer">
  Page number
</ParamField>

<ParamField query="limit" type="integer">
  Number of products per page
</ParamField>

<ParamField query="search" type="string">
  Search products by name
</ParamField>

<ParamField query="category_id" type="integer">
  Filter by category ID
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="data" type="object">
  <ResponseField name="products" type="array">
    <ResponseField name="id" type="string">
      Product ID (UUID)
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Product name
    </ResponseField>
    
    <ResponseField name="price" type="number">
      Product price
    </ResponseField>
    
    <ResponseField name="stock" type="integer">
      Available stock
    </ResponseField>
    
    <ResponseField name="category_id" type="integer">
      Category ID
    </ResponseField>
    
    <ResponseField name="category_name" type="string">
      Category name
    </ResponseField>
    
    <ResponseField name="image_url" type="string">
      Product image URL
    </ResponseField>
    
    <ResponseField name="deleted_at" type="string">
      Soft delete timestamp (null if active)
    </ResponseField>
  </ResponseField>
  
  <ResponseField name="pagination" type="object">
    Pagination information
  </ResponseField>
</ResponseField>

### Example

```bash
curl -X GET "https://localhost:8080/api/v1/products?page=1&limit=10&search=coffee" \
  -H "Cookie: access_token=YOUR_TOKEN"
```

<ResponseExample>
```json
{
  "message": "Products retrieved successfully",
  "data": {
    "products": [
      {
        "id": "123e4567-e89b-12d3-a456-426614174000",
        "name": "Espresso Coffee",
        "price": 25000,
        "stock": 100,
        "category_id": 1,
        "category_name": "Beverages",
        "image_url": "https://storage.example.com/products/espresso.jpg",
        "deleted_at": null
      }
    ],
    "pagination": {
      "current_page": 1,
      "per_page": 10,
      "total_data": 1,
      "total_page": 1
    }
  }
}
```
</ResponseExample>

## Create Product

<CodeBlock title="POST /api/v1/products">
Create a new product with multiple options.

**Authentication Required:** Yes

**Roles:** `admin`, `manager`
</CodeBlock>

### Request Body

<ParamField body="name" type="string" required>
  Product name (3-100 characters)
</ParamField>

<ParamField body="price" type="number" required>
  Product price
</ParamField>

<ParamField body="cost_price" type="number" required>
  Cost price (minimum 0)
</ParamField>

<ParamField body="stock" type="integer" required>
  Initial stock (minimum 0)
</ParamField>

<ParamField body="category_id" type="integer" required>
  Category ID
</ParamField>

<ParamField body="options" type="array">
  Array of product options
  
  <ParamField body="name" type="string" required>
    Option name (1-100 characters)
  </ParamField>
  
  <ParamField body="additional_price" type="number">
    Additional price for this option (minimum 0)
  </ParamField>
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="data" type="object">
  Created product information
</ResponseField>

### Example

```bash
curl -X POST https://localhost:8080/api/v1/products \
  -H "Content-Type: application/json" \
  -H "Cookie: access_token=YOUR_TOKEN" \
  -d '{
    "name": "Cappuccino",
    "price": 30000,
    "cost_price": 15000,
    "stock": 50,
    "category_id": 1,
    "options": [
      {
        "name": "Extra Shot",
        "additional_price": 5000
      },
      {
        "name": "Oat Milk",
        "additional_price": 8000
      }
    ]
  }'
```

## Update Product

<CodeBlock title="PATCH /api/v1/products/{id}">
Update details of a specific product by its ID.

**Authentication Required:** Yes

**Roles:** `admin`, `manager`
</CodeBlock>

### Path Parameters

<ParamField path="id" type="string" required>
  Product ID (UUID format)
</ParamField>

### Request Body

<ParamField body="name" type="string">
  Product name (3-100 characters)
</ParamField>

<ParamField body="price" type="number">
  Product price
</ParamField>

<ParamField body="cost_price" type="number">
  Cost price (minimum 0)
</ParamField>

<ParamField body="stock" type="integer">
  Stock quantity (minimum 0)
</ParamField>

<ParamField body="category_id" type="integer">
  Category ID
</ParamField>

<ParamField body="change_type" type="string">
  Stock change type: `sale`, `restock`, `correction`, `return`, or `damage`
</ParamField>

<ParamField body="note" type="string">
  Stock change note (max 255 characters)
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="data" type="object">
  Updated product information
</ResponseField>

### Example

```bash
curl -X PATCH https://localhost:8080/api/v1/products/223e4567-e89b-12d3-a456-426614174001 \
  -H "Content-Type: application/json" \
  -H "Cookie: access_token=YOUR_TOKEN" \
  -d '{
    "price": 32000,
    "stock": 75,
    "change_type": "restock",
    "note": "Weekly restock"
  }'
```

## Get Stock History

<CodeBlock title="GET /api/v1/products/{id}/stock-history">
Get stock history for a specific product with pagination.

**Authentication Required:** Yes

**Roles:** `admin`, `manager`
</CodeBlock>

### Path Parameters

<ParamField path="id" type="string" required>
  Product ID (UUID format)
</ParamField>

### Query Parameters

<ParamField query="page" type="integer">
  Page number
</ParamField>

<ParamField query="limit" type="integer">
  Number of records per page
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="data" type="object">
  <ResponseField name="history" type="array">
    <ResponseField name="id" type="string">
      History record ID
    </ResponseField>
    
    <ResponseField name="product_id" type="string">
      Product ID
    </ResponseField>
    
    <ResponseField name="change_type" type="string">
      Type of stock change
    </ResponseField>
    
    <ResponseField name="change_amount" type="integer">
      Amount changed
    </ResponseField>
    
    <ResponseField name="previous_stock" type="integer">
      Stock before change
    </ResponseField>
    
    <ResponseField name="current_stock" type="integer">
      Stock after change
    </ResponseField>
    
    <ResponseField name="note" type="string">
      Change note
    </ResponseField>
    
    <ResponseField name="created_by" type="string">
      User who made the change
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Change timestamp
    </ResponseField>
  </ResponseField>
  
  <ResponseField name="pagination" type="object">
    Pagination information
  </ResponseField>
</ResponseField>

### Example

```bash
curl -X GET "https://localhost:8080/api/v1/products/223e4567-e89b-12d3-a456-426614174001/stock-history?page=1&limit=10" \
  -H "Cookie: access_token=YOUR_TOKEN"
```
