## List Orders

<CodeBlock title="GET /api/v1/orders">
Get a list of orders with filtering by status and user.

**Authentication Required:** Yes

**Roles:** `admin`, `manager`, `cashier`
</CodeBlock>

### Query Parameters

<ParamField query="page" type="integer">
  Page number
</ParamField>

<ParamField query="limit" type="integer">
  Number of orders per page
</ParamField>

<ParamField query="statuses" type="array">
  Order statuses to filter: `open`, `in_progress`, `served`, `paid`, `cancelled`
</ParamField>

<ParamField query="user_id" type="string">
  Filter by User ID (UUID format)
</ParamField>

### Response

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="data" type="object">
  <ResponseField name="orders" type="array">
    <ResponseField name="id" type="string">
      Order ID (UUID)
    </ResponseField>
    
    <ResponseField name="queue_number" type="string">
      Queue/order number
    </ResponseField>
    
    <ResponseField name="type" type="string">
      Order type: `dine_in` or `takeaway`
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Order status
    </ResponseField>
    
    <ResponseField name="is_paid" type="boolean">
      Payment status
    </ResponseField>
    
    <ResponseField name="net_total" type="integer">
      Total amount
    </ResponseField>
    
    <ResponseField name="user_id" type="string">
      User who created the order
    </ResponseField>
    
    <ResponseField name="items" type="array">
      Order items
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Creation timestamp
    </ResponseField>
  </ResponseField>
  
  <ResponseField name="pagination" type="object">
    Pagination information
  </ResponseField>
</ResponseField>

### Example

```bash
curl -X GET "https://localhost:8080/api/v1/orders?page=1&limit=10&statuses=open&statuses=in_progress" \
  -H "Cookie: access_token=YOUR_TOKEN"
```

## Create Order

<CodeBlock title="POST /api/v1/orders">
Create a new order with multiple items.

**Authentication Required:** Yes

**Roles:** `admin`, `manager`, `cashier`
</CodeBlock>

### Request Body

<ParamField body="type" type="string" required>
  Order type: `dine_in` or `takeaway`
</ParamField>

<ParamField body="items" type="array" required>
  Array of order items (minimum 1 item)
  
  <ParamField body="product_id" type="string" required>
    Product ID (UUID)
  </ParamField>
  
  <ParamField body="quantity" type="integer" required>
    Quantity
  </ParamField>
  
  <ParamField body="options" type="array">
    Product options/variants
    
    <ParamField body="product_option_id" type="string" required>
      Product option ID
    </ParamField>

</ParamField>

  Success message
</ResponseField>

<ResponseField name="data" type="object">
  Created order details
</ResponseField>

```bash
curl -X POST https://localhost:8080/api/v1/orders \
  -H "Content-Type: application/json" \
```
